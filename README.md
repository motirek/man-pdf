# man-pdf
Script to transfer MAN of command to .PDF file

    man-pdf [-h] [-c p s] -- Program to transfer man of command to .pdf file

         -h   Show help
         -c   Command for man (will be used as name for new pdf)
         -p   PDF path (default: /home/miramadera/ManPDF)
         -z   Show in Zathura
